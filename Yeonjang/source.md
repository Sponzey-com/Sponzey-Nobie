# source.md

## 역할

- `Yeonjang`은 Nobie core 밖에서 권한 작업과 장치 작업을 수행하는 Rust 확장 런타임입니다.

## 주요 책임

- MQTT로 Nobie에 연결
- 카메라, 화면, 마우스, 키보드, 명령 실행, 앱 실행 같은 장치/시스템 메서드 제공
- 마우스와 키보드는 액션 기반 요청도 받아들일 수 있는 추상화 계약 제공
- 연결, 연장 정보, 권한을 설정하는 네이티브 GUI 제공
- 연결이 끊긴 뒤에도 GUI에서 바로 재접속할 수 있는 재연결 흐름 제공

## 메모

- Nobie는 Yeonjang을 단순 헬퍼 라이브러리가 아니라 실제 실행 주체로 다룹니다.
- 캡처처럼 바이너리 비중이 큰 결과물도 Yeonjang을 거쳐 MQTT로 돌아옵니다.
